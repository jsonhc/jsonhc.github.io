关于域名更换的公告

2026 年 1 月 24 日更新

经实测，使用 Cloudflare CDN 加速 GitHub Pages 站点效果在国内貌似不太好，并且也要将主域名改为个人引导页面。

目前已从 GitHub Pages 转移到 Vercel 新加坡区部署，域名也改为了 blog.dorahonor.me，实测访问速度有显著提升。

原有的根域名 dorahonor.me 在 2 月初之前仍为博客页，但建议及时更新收藏夹中的链接。

根域名 dorahonor.me 将在 2 月初左右换成个人引导页。欢迎访问！

感谢大家一直以来的支持与关注！
